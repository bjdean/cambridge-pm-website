On Wednesday 02 Jul 2003 1:57 am, Lucy McWilliam wrote:
> #!/usr/bin/perl -w
> use strict;
> my ($line);
>
> open(FILE, "file");
> while (<FILE>) {
>     $line = <FILE>;
>     if ($line =~ /^\>/) {
>        print $line;
>    }
> }
>
> If a file contains the sequence data below, what is it about $line  =
> <FILE> that makes the script above not print out the lines starting with a
> '>' ?

The <FILE> in while(<FILE>) reads in a line, the <FILE> in $line=<FILE> reads 
in the next. So $line only ever gets to be every other line, only the lines 
without the >.
