I tried this and it worked ok for me. Watch out for the difference
betwen getopt & getopts. Code something like this:

#! /usr/bin/perl -w

use strict;
use Getopt::Std;

my %opts = ();

getopts('oD:I', \%opts);

foreach (keys %opts) {
    print "$_ : $opts{$_}\n";
}

Gives output of:

[simonw@neo perl]$ perl getopt -o -I -D three
o : 1
I : 1
D : three
