> > Also you should know that system("string with spaces") is equivalent
> > to system(qw(string with spaces)) .

 No, it isn't.

 qw(string with spaces) is a list, so system(qw(string with spaces))
is actually equivalent to system('string', 'with', 'spaces').

 How is this different? Well, if you call system with the single
string, shell-like processing is done on it, whereas if you call
it with a list, the arguments are passed through literally.

 Compare:  perl -e 'system("echo *")'
     and:  perl -e 'system("echo", "*")'

 The other difference is that a double-quoted string does variable
interpolation, whereas qw doesn't.
