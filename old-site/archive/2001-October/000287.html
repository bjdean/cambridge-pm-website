I'm probably suffering from brain failure, here, but I'm looking for an
elegant way to do turn something like:

$before = { foo => [ $a, $b, $c ], bar => [ $d, $e, $f ], ... }

into

[ $a, $b, $c, $d, $e, $f ... ]

The size of each array reference may vary.  Obviously, I can do
something like:

$after = [];

foreach (values %$before) {
    push @$after, @$_;
}

But I was wondering if there was a more elegant way that doesn't involve
so much data movement (I suspect the above could get very slow if
$before is a large data structure).

Any ideas?
