Of course, the /much/ easier way to do this is to install LILO to
the linux root partition, as opposed to the MBR. This way, after you
install Windows, you can use DOS fdisk to set the linux root
partition as active, and you'll get LILO back.

Here's a picture:       +------+
                      .-+ Hurd |
                     /  +------+
                  +-+----+   +-------+
              o---+ LILO +---+ Lunix |
+-----+     /     +-+----+   +-------+
| MBR +---o          \
+-----+     \     +---+---------+
              o---+ DOS / NTLDR |
                  +-------------+

Thus adding your secondary or tertiary OS is just as easy
irrespective of the order you do it in. DYS?
