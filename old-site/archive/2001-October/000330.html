Ah.  This seems to work:

#!/usr/bin/perl -w

while (<DATA>) {
    my (@a) = /(\"(?:\\.|[^\"])*\"|[^,\n]*)[,\n]/g;
    map  { s/^\"(.*)\"$/$1/; s/\\\"/\"/g; } @a;
    print join("\n", @a, '');
}

__DATA__
Wibble,12345,,"Foo, Bar","\"The cat sat on the mat\", she said"

i.e. every non-quoted field *has* to be terminated either with a comma
or a new line.
