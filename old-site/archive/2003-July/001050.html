[ Bit quiet in here, isn't it? ]

Assume I have an array containing data, an (equally-sized) array containing
whether each data point is valid or not and I wish to select all
elements from the data for which the selector is true

eg from 

@data = ( 1, 2, 3, 4 )
@selector = ( 0, 1, 1, 0 )

I wish to obtain @valid = ( 2, 3 )

Obviously there are lots of ways of doing this, but are there any which
don't involve use of a temporary variable (or trashing the contents of
@data or @valid)? The closest I can get is

{
  my $i = 0;
  @valid = map { $_->[0] } grep { $_->[1] } map { [$_, $selector[$i++]] } @data;
}

use Language::Functional also springs to mind...
