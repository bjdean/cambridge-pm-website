Cookies being the important thing. People have basically abandoned HTTP
authentication for professional websites because they can't override its
user interface in browsers.

> (this was the early implementation I saw of Pete's 8 or 9 months ago)

Yes, that arose from an all-nighter :) I'd dug it up, removed the
reimplementation of HTML::Form, changed it from a hideous state machine
into a reasonable language, and made it follow redirects transparently.

my $wex = new Wex();

# Go to a page (similar to javascript).
$wex->location("http://www.orange.net/");

# Set a couple of inputs, as if they were typed in.
$wex->input('usr', $USER);
$wex->input('pwd', $PASSWORD);

# Click on the first submit button found in the form last
# touched. Note that there are two forms on the orange page,
# the first of which is a search one which we don't want.
$wex->click();

etc.
