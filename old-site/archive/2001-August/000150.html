Note that

   '}_).} "*}_} }./}(}_ }}_, (}_+}_,*"'   &
 ~ '/-  +   *,+ <  + :- /:-   << :-  @'   |
   '  ``   @     `` `   @  ` `  `   '

evaluates to

  print "Just another Perl hacker,
  "

by bitwise string operations.

And (?{expr}) in a regexp is a zero-width assertion that evaluates the code
(and always succeeds).
