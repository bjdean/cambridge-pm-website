Which version of Perl are you using? I seem to remember that the version
of Getopt::Std whch fills in a hash is a relatively new thing (one of the
5.005 series perhaps).

And if you can't use "our" because you're not using 5,6.x, then the 
alternative is to use "use vars" to predeclare your package variables.

use vars qw($a $set %of @variables);

Well there's Getopt::Long - but that's more complicated.

Perhaps you should just download and install a more up to date version
of Getopt::Std.
