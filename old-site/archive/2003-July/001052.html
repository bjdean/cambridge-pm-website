> Assume I have an array containing data, an (equally-sized)
> array containing whether each data point is valid or not 
> and I wish to select all elements from the data for which 
> the selector is true
>
> eg from 
>
> @data = ( 1, 2, 3, 4 )
> @selector = ( 0, 1, 1, 0 )
>
> I wish to obtain @valid = ( 2, 3 )

Something like this perhaps:

#!/usr/local/bin/perl

use strict;
use warnings;

my @data = ( 1, 2, 3, 4 );
my @selector = ( 0, 1, 1, 0 );

my @valid = @data[ grep { $selector[$_] } 0 .. $#selector ];

print "@valid\n";
