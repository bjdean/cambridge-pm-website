Another alternative is to use something like Win4Lin or VMWare to run
Windows under Linux (or the other way around, with VMWare, which can do
both).  Costs a bit of money, but it's a solution I use at home.

Games tend not to run well or at all under Windows this way, but if
that's acceptable, this solution works quite well.  I use Win4Lin at
home to run MS Office and IE, and it works a treat.  Costs about £50
from Amazon.  VMWare is more expensive (and slower), but more flexible.
