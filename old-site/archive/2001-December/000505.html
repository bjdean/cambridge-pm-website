Depends how much you want to run each OS, but one approach not mentioned
is to use a bootable cdrom with linux on it. I think that most
commercial distros come with one. No permission needed :-)

BTW2 - if your disk has working Linux, and you have made a DOS partition
with the linux fdisk, then MSW will install into that one. You'll need
to make a LILO floppy beforehand, since your boot sector will be
blatted.

I did this, and found the boot-floppy-to-boot-linux sufficiently
convenient (only adds a few seconds to boot time) that I never did get
round to making a magic LILO boot. Plus, you remove the floppy, and it's
a pure windows machine, honest.
